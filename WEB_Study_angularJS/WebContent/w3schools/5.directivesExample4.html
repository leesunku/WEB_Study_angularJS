Repeating HTML Elements

<div ng-app="" ng-init="names = [ {name : 'Jani', country : 'Norway'}, {name : 'Hege', country : 'Sweden'}, {name : 'Kai', country : 'Denmark'} ]"> <ul> <li ng-repeat="x in names"> {{ x.name + ', ' + x.country }} </li> </ul> </div>
  • {{ x.name + ', ' + x.country }}
예제 설명 :
앙귤라는 완벽한 데이터 베이스 CRUD(Create, Read, Update, Delete) 어플리케이션이다.

The ng-app Directive

ng-app directives는 앙귤라 어플리케이션에 root 엘리먼트를 정의 한다.
웹페이지가 로드되면, 어플리케이션에서 자동으로 초기화된다.

The ng-init Directive

ng-init directives는 앙귤라 어플리케이션에 초기화 값을 의미한다.
일반적으로, ng-init를 사용하지 않을 것이다.대신 컨트롤러나 모듈을 사용할 것이다.

The ng-model Directive

ng-model directives는 어플리케이션 데이터를 html 컨트롤(input, select, textarea)에 값을 바인딩한다.
