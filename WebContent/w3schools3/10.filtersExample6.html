AngularJS Example

Filter an Array Based on User Input

<div ng-app="skApp" ng-controller="skCtrl"> <p> 글자 입력 : </p> <input type="text" ng-model="test"> <ul> <li ng-repeat = "x in names | filter:test"> {{ x }} </li> </ul> </div> <script type="text/javascript"> angular.module('skApp', []).controller('skCtrl', function($scope){ $scope.names = [ '이선구', '홍길동', '전우치', '이순신', '세종대왕', '유관순' ]; }); </script>

글자 입력 :

  • {{ x }}
예제 설명 :
ng-model로 설정한 input 필드에, 우리가 input 필드에 입력한 값을 사용할 수 있다.
