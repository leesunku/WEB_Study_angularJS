angular.toJson - 사파리 브라우져에서는 date 오브젝트에 잘못된 date 값이 있다면, json으로 받을 때, RangeError 를 통해 null이 리턴된다. 이 것을 예방하기 위한 유일한 방법은 Date.prototype.toJSON을 monkeypatch 이다. 방법은 다음과 같다. var _DatetoJSON = Date.prototype.toJSON; Date.prototype.toJSON = function() { try { return _DatetoJSON.call(this); } catch(e) { if (e instanceof RangeError) { return null; } throw e; } }; 관한 정보는 https://github.com/angular/angular.js/pull/14221 을 봅니다. angular.element - 자스민 버젼 1.x 은 angular.element를 사용할 수 없다. ngClass - class 속성 값을 보간할 수 없다. ngClass에 지정된 엘리먼트가 같을 경우에 사용한다. ngStyle - style 속성 값을 보간할 수 없다. ngStyle에 지정된 엘리먼트가 같을 경우에 사용한다. 이게 아닌거 같아, 번역하는 것도 아니고 ㅠ w3스쿨로 가자
