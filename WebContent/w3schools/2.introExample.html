AngularJS Example

AngularJS Extends HTML

<div ng-app=""> <p>Name: <input type="text" ng-model="name"></p> <p ng-bind="name"></p> </div>

Name:

예제 설명 :
웹페이지가 로드되면, 앙귤라가 자동으로 시작됩니다.
ng-app - div 태그 는 알귤라 앱으로 지정합니다.
ng-model - input 필드에서 name 키의 값을 바인딩 합니다.
ng-bind - name 키에 값을 innerHTML(html에 보여주다.), p태그에 바인딩합니다.
